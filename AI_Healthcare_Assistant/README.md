# AI Healthcare Assistant

An AI-powered healthcare application with disease prediction, skin disease detection, voice assistance, medicine reminders, and emergency alerts.

## Project Structure

- **app/frontend/**: Streamlit-based frontend components
- **app/backend/**: FastAPI backend services and routes
- **app/database/**: Database models and configuration
- **models/**: Pre-trained ML models for various healthcare tasks
- **datasets/**: Training and test datasets
- **nlp/**: Natural Language Processing modules
- **reminders/**: Medicine reminder scheduling system
- **utils/**: Utility functions and helpers
- **config/**: Configuration and settings
- **tests/**: Unit and integration tests

## Features

- Disease Prediction using ML models
- Skin Disease Detection with computer vision
- Voice-based Health Assistant
- Medicine Reminder System
- Emergency Alert System
- Health History Tracking
- Multi-language support (Nepali)
